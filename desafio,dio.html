Nas primeira aulas foi explicado sobre a parte teorica do site e html, tendo enfase na parte funcional, diferenciando trabalho do servidor para do cliente, explanando termos muito usados mas poucos entendidos como.
  • Front-end: parte do site que o usuario ve, ou seja, a interface.
  • Back-end: parte do site que o usuario nao ve, ou seja, o servidor.
  • Full-stack: pessoa que trabalha tanto com front-end quanto com back-end.
  • HTML: linguagem de marcação usada para estruturar o conteudo na web.
  • CSS: linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML.
  • JavaScript: linguagem de programação usada para criar interatividade e dinamismo em páginas web.
  • Frameworks: conjuntos de ferramentas e bibliotecas que facilitam o desenvolvimento web, como React, Angular e Vue.js para front-end, e Node.js, Django e Ruby on Rails para back-end.
  • Responsive Design: abordagem de design que visa criar sites que se adaptam a diferentes tamanhos de tela e dispositivos.
  • SEO (Search Engine Optimization): conjunto de práticas para melhorar a visibilidade de um site nos motores de busca.
  • UX (User Experience): foco na experiência do usuário ao interagir com um site ou aplicação.
